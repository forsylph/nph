# 재배치-실행계획

## 1. 목적

이 문서는 재배치 작업의 실행 계획과 적용 결과를 짧게 남기는 문서다.

## 2. 적용 결과 요약

- 기존 문서는 `old Data`로 보존
- `03.analysis_results`에는 새 기준 구조 생성
- `039` 계열 리팩토링 문서는 `95/953`로 분리
- `030.index`는 허브, `031~033`은 기반기술, `035`는 도메인, `037`은 trace, `038`은 검증/참고로 정리

## 3. 후속 우선순위

1. `033.platform-services` 상세화
2. `037.runtime-trace`와 `95/953` 링크 보강
3. `030.index`와 `035`의 업무/기술 연결 강화
