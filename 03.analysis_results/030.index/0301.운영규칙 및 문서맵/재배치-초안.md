# 재배치-초안

## 1. 목적

이 문서는 `03.analysis_results` 재구성의 초안 메모를 보존하는 문서다.
현재는 실행 완료 후 참고 메모 성격으로만 유지한다.

## 2. 현재 해석

- `031.front-channel`: 사용자와 가까운 프론트 진입 구조
- `032.framework-core`: DevOn 중심의 서버 코어 실행 구조
- `033.platform-services`: DevOn 외부 솔루션 및 패키지
- `035.Biz-medical-Domain`: 의료업무/도메인 관점 문서
- `037.runtime-trace`: 실제 사례 추적
- `038.fact-todo-reference`: 근거/검증/미확인 관리
- `95/953.refactoring-ideation`: 개선안과 리팩토링 아이데이션

## 3. 메모

현재 기준본 운영은 `README.md`와 `운영규칙.md`를 우선한다.
