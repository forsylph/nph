# DevOn Frameworkмқҳ лҸ…м°Ҫм„ұ л¶„м„қ

> Struts 1.xлҘј кё°л°ҳмңјлЎң н•ҳл©ҙм„ңлҸ„ MiPlatform м—°лҸҷкіј м—”н„°н”„лқјмқҙмҰҲ нҠ№нҷ”лҘј мң„н•ҙ м¶”к°Җлҗң DevOnл§Ңмқҳ лҸ…м°Ҫм Ғ кё°лҠҘ
> мЈјмқҳ: ліёл¬ём—җлҠ” м•„нӮӨн…ҚмІҳ м„ӨлӘ…мҡ© к°ҖмғҒ мҪ”л“ңк°Җ нҸ¬н•Ёлҗ©лӢҲлӢӨ. нҳ„мһ¬ л°ұм—…м…Ӣм—җм„ң м§Ғм ‘ нҷ•мқёлҗң н•ӯлӘ©мқҖ `MiplatformServlet`, `GeneralServlet`, `MiplatformRequest/Response/Converter`, `AbstractMiplatformCommand`, `devon-framework.xml`, `requisite.xml` кё°мӨҖмһ…лӢҲлӢӨ.

## к°ңмҡ”

**DevOn = Struts 1.x + MiPlatform м—°лҸҷ + м—”н„°н”„лқјмқҙмҰҲ нҷ•мһҘ**

DevOnмқҖ Struts 1.xмқҳ м•„нӮӨн…ҚмІҳлҘј л”°лҘҙл©ҙм„ңлҸ„, лӢӨмқҢмқҳ лҸ…м°Ҫм Ғ кё°лҠҘмқ„ м¶”к°Җн–ҲмҠөлӢҲлӢӨ:

| кө¬л¶„ | Struts 1.x | DevOnмқҳ лҸ…м°Ҫм„ұ |
|------|------------|----------------|
| **View м—°лҸҷ** | JSP | **MiPlatform Dataset м—°лҸҷ** |
| **л№„мҰҲлӢҲмҠӨ кі„мёө** | Action + Service | **Process Component (PC)** |
| **лҚ°мқҙн„° м ‘к·ј** | JDBC / DAO | **XML Query кё°л°ҳ м ‘к·ј кі„мёө** |
| **кІҖмҰқ** | ActionForm.validate() | **Requisite (м„ м–ём Ғ кІҖмҰқ)** |
| **нҠёлһңмһӯм…ҳ** | мҲҳлҸҷ кҙҖлҰ¬ | **TxServiceUtil (м„ м–ём Ғ нҠёлһңмһӯм…ҳ)** |
| **лҚ°мқҙн„° ліҖнҷҳ** | мҲҳлҸҷ | **MiplatformConverter (мһҗлҸҷ ліҖнҷҳ)** |

## 2. Process Component (PC) кі„мёө

### 2.1 EJB лҢҖмІҙ кө¬мЎ°

Struts 1.xлҠ” Actionмқҙ м§Ғм ‘ Service/DAOлҘј нҳём¶ңн•ҳм§Җл§Ң, DevOnмқҖ **Process Component (PC)** кі„мёөмқ„ м¶”к°Җн–ҲмҠөлӢҲлӢӨ.

```
Struts 1.x:
Action вҶ’ Service вҶ’ DAO вҶ’ DB

DevOn:
Command вҶ’ PC (Process Component) вҶ’ XML Query м ‘к·ј кі„мёө вҶ’ DB
          вҶ“
      TxServiceUtil (нҠёлһңмһӯм…ҳ)
```

```java
            "app/emp/login.xml",  // XML нҢҢмқј кІҪлЎң
            "retrieveUserInfo",    // мҝјлҰ¬ мқҙлҰ„
```

```java
        // нҠёлһңмһӯм…ҳ л§ӨлӢҲм Җ мғқм„ұ
        TransactionManager txManager = /* м„Өм • кё°л°ҳ transaction manager мҳҲмӢң */ null;
```

## 3. XML Query (к°ңл…җ лӘЁлҚё)

## 4. Requisite (м„ м–ём Ғ кІҖмҰқ)

### 4.1 XML кё°л°ҳ мһ…л Ҙк°’ кІҖмҰқ

Struts 1.xлҠ” ActionFormмқҳ validate() л©”мҶҢл“ңлҘј мӮ¬мҡ©н•ҳм§Җл§Ң, DevOnмқҖ **XML кё°л°ҳ м„ м–ём Ғ кІҖмҰқ**мқ„ кө¬нҳ„н–ҲмҠөлӢҲлӢӨ.

```xml
<!-- devonhome/conf/requisite.xml -->
<requisites>
    <requisite name="LoginUser">
        <field name="usid" required="true" maxLength="20">
            <message>мӮ¬мҡ©мһҗ IDлҠ” н•„мҲҳмһ…лӢҲлӢӨ.</message>
        </field>
        <field name="passwd" required="true" minLength="6">
            <message>л№„л°ҖлІҲнҳёлҠ” 6мһҗ мқҙмғҒмқҙм–ҙм•ј н•©лӢҲлӢӨ.</message>
        </field>
    </requisite>

    <requisite name="SavePatient">
        <field name="pid" required="true" type="number"/>
        <field name="name" required="true" maxLength="50"/>
        <field name="birthDate" required="true" type="date"/>
    </requisite>
</requisites>
```

```java
// Commandм—җм„ң мһҗлҸҷ кІҖмҰқ
public abstract class AbstractMiplatformCommand {

    public AbstractMiplatformCommand() {
        // мғқм„ұмһҗм—җм„ң мһҗлҸҷ кІҖмҰқ мӢӨн–ү
        validateByRequisite(data);
    }

    protected void validateByRequisite(LData data) throws Exception {
        // requisite.xmlм—җм„ң н•ҙлӢ№ Actionмқҳ кІҖмҰқ к·ңм№ҷ лЎңл“ң
        Requisite requisite = RequisiteLoader.load(actionName);

        // к°Ғ н•„л“ң кІҖмҰқ
        for (Field field : requisite.getFields()) {
            String value = data.getString(field.getName());

            // н•„мҲҳ кІҖмҰқ
            if (field.isRequired() && (value == null || value.isEmpty())) {
                throw new ValidationException(field.getMessage());
            }

            // кёёмқҙ кІҖмҰқ
            if (field.getMaxLength() > 0 && value.length() > field.getMaxLength()) {
                throw new ValidationException(field.getName() + " кёёмқҙ мҙҲкіј");
            }

            // нғҖмһ… кІҖмҰқ
            if ("number".equals(field.getType())) {
                try {
                    Integer.parseInt(value);
                } catch (NumberFormatException e) {
                    throw new ValidationException(field.getName() + "лҠ” мҲ«мһҗм—¬м•ј н•©лӢҲлӢӨ.");
                }
            }
        }
    }
}
```

**лҸ…м°Ҫм„ұ:**
- Java мҪ”л“ңк°Җ м•„лӢҢ **XMLлЎң кІҖмҰқ к·ңм№ҷ м •мқҳ**
- Command мғқм„ұ мӢң **мһҗлҸҷ мӢӨн–ү**
- мһ¬мӮ¬мҡ© к°ҖлҠҘн•ң кІҖмҰқ к·ңм№ҷ

## 5. Interceptor Chain

### 5.1 AOP мҠӨнғҖмқј кіөнҶө мІҳлҰ¬

Struts 1.xлҠ” RequestProcessorлҘј нҷ•мһҘн•ҙм•ј н•ҳм§Җл§Ң, DevOnмқҖ **Interceptor Chain**мқ„ кө¬нҳ„н–ҲмҠөлӢҲлӢӨ.

```xml

<interceptor-stack name="defaultStack">
    <interceptor-ref name="loginCheck"/>
    <interceptor-ref name="converter"/>
    <interceptor-ref name="command"/>
</interceptor-stack>

<interceptor-stack name="notLoginCheckStack">
    <interceptor-ref name="converter"/>
    <interceptor-ref name="command"/>
</interceptor-stack>
```

```java
// Interceptor кө¬нҳ„ к°ңл…җ мҳҲмӢң
// мӢӨм ң л°ұм—…м…Ӣм—җм„ң нҷ•мқёлҗң LoginCheckInterceptorлҠ” LAbstractInterceptor кё°л°ҳмқҙлӢӨ.
public class LoginCheckInterceptor /* conceptual */ {

    public String doIntercept(ActionInvocation invocation) throws Exception {
        // м„ём…ҳ мІҙнҒ¬
        HttpSession session = LActionContext.getHttpServletRequest().getSession();
        if (session.getAttribute("userInfo") == null) {
            return "login_required";
        }

        // лӢӨмқҢ Interceptor лҳҗлҠ” Command мӢӨн–ү
        return invocation.invoke();
    }
}

// мЈјмқҳ: м•„лһҳ ConverterInterceptorлҠ” м„ӨлӘ…мҡ© мҳҲмӢңлӢӨ.
// нҳ„мһ¬ л°ұм—…м…Ӣм—җм„ңлҠ” ConverterInterceptor / CommandInterceptor мӢӨнҢҢмқјмқ„ м§Ғм ‘ нҷ•мқён•ҳм§Җ лӘ»н–ҲлӢӨ.
public class ConverterInterceptor /* conceptual */ {
    public String doIntercept(ActionInvocation invocation) throws Exception {
        // MiPlatform Dataset вҶ’ LData ліҖнҷҳ
        convertRequestData();

        String result = invocation.invoke();

        // LData вҶ’ MiPlatform Dataset ліҖнҷҳ
        convertResponseData();

        return result;
    }
}
```

- Springмқҳ AOPмҷҖ мң мӮ¬н•ң **Interceptor Chain**

## 6. ActionContext (ThreadLocal кё°л°ҳ м»Ён…ҚмҠӨнҠё)

### 6.1 мҡ”мІӯ-мқ‘лӢө к°қмІҙ мӨ‘м•ҷ кҙҖлҰ¬

```java
// м»Ён…ҚмҠӨнҠё кҙҖлҰ¬ к°ңл…җ мҳҲмӢң
```

## 8. к°ңмқём •ліҙ ліҙнҳёлҘј мң„н•ң мһҗлҸҷ л§ҲмҠӨнӮ№

### 8.1 лЎңк№… мӢң к°ңмқём •ліҙ л§ҲмҠӨнӮ№

```java
// DevOnмқҳ лҸ…м°Ҫм Ғ ліҙм•Ҳ кё°лҠҘ
public abstract class AbstractMiplatformCommand {

    protected void logWrite() {
        LData data = getParamData();

        // к°ңмқём •ліҙ н•„л“ң мһҗлҸҷ л§ҲмҠӨнӮ№
        String maskedData = maskPersonalInfo(data);

        // л§ҲмҠӨнӮ№лҗң лҚ°мқҙн„°л§Ң лЎңк·ём—җ кё°лЎқ
        logger.info("Request Data: " + maskedData);
    }

    private String maskPersonalInfo(LData data) {
        // мЈјлҜјл“ұлЎқлІҲнҳё л§ҲмҠӨнӮ№
        if (data.containsKey("ssn")) {
            String ssn = data.getString("ssn");
            data.setString("ssn", ssn.substring(0, 6) + "-*******");
        }

        // н•ёл“ңнҸ°лІҲнҳё л§ҲмҠӨнӮ№
        if (data.containsKey("phone")) {
            String phone = data.getString("phone");
            data.setString("phone", phone.replaceAll("(\d{3})-(\d{3,4})-(\d{4})", "$1-****-$3"));
        }

        return data.toString();
    }
}
```

**лҸ…м°Ҫм„ұ:**
- **мһҗлҸҷ к°ңмқём •ліҙ л§ҲмҠӨнӮ№**
- Command мғқм„ұ мӢң мһҗлҸҷ мӢӨн–ү
- мқҳлЈҢ мӢңмҠӨн…ң нҠ№нҷ” кё°лҠҘ

## мҡ”м•Ҫ: DevOnмқҳ лҸ…м°Ҫм„ұ

### н•өмӢ¬ лҸ…м°Ҫм Ғ кё°лҠҘ

| кё°лҠҘ | м„ӨлӘ… | Struts 1.x лҢҖл№„ |
|------|------|-----------------|
| **Dataset м—°лҸҷ** | MiPlatform Dataset вҶ” LData мһҗлҸҷ ліҖнҷҳ | JSP лҢҖмӢ  RIA м—°лҸҷ |
| **Process Component** | EJBлҘј лҢҖмІҙн•ҳлҠ” кІҪлҹү л№„мҰҲлӢҲмҠӨ кі„мёө | Service лҢҖмІҙ |
| **XML Query** | SQLмқ„ XMLлЎң мҷ„м „ л¶„лҰ¬ | DAO н•ҳл“ңмҪ”л”© лҢҖмІҙ |
| **Requisite** | XML кё°л°ҳ м„ м–ём Ғ кІҖмҰқ | validate() лҢҖмІҙ |
| **TxServiceUtil** | м„ м–ём Ғ нҠёлһңмһӯм…ҳ кҙҖлҰ¬ | мҲҳлҸҷ нҠёлһңмһӯм…ҳ лҢҖмІҙ |
| **Interceptor Chain** | AOP мҠӨнғҖмқј кіөнҶө мІҳлҰ¬ | RequestProcessor лҢҖмІҙ |
| **ActionContext** | ThreadLocal кё°л°ҳ м»Ён…ҚмҠӨнҠё | нҢҢлқјлҜён„° м „лӢ¬ к°җмҶҢ |
| **кі„мёөм Ғ Navigation** | м—…л¬ҙлі„ м„Өм • нҢҢмқј л¶„лҰ¬ | лӢЁмқј нҢҢмқј лҢҖмІҙ |
| **мһҗлҸҷ л§ҲмҠӨнӮ№** | к°ңмқём •ліҙ ліҙнҳё лЎңк№… | м—ҶмқҢ |

### DevOnмқҙ н•ҙкІ°н•ң л¬ём ң

```
1. RIA(MiPlatform) м—°лҸҷ л¬ём ң
   вҶ’ Dataset/LData ліҖнҷҳ кі„мёөмңјлЎң н•ҙкІ°

2. EJBмқҳ ліөмһЎм„ұ
   вҶ’ PC (Process Component)лЎң кІҪлҹүнҷ”

3. SQL кҙҖлҰ¬ л¬ём ң
   вҶ’ XML QueryлЎң SQL л¶„лҰ¬

4. кІҖмҰқ мҪ”л“ң мӨ‘ліө
   вҶ’ RequisiteлЎң м„ м–ём Ғ кІҖмҰқ

5. нҠёлһңмһӯм…ҳ кҙҖлҰ¬ ліөмһЎм„ұ
   вҶ’ TxServiceUtilлЎң лӢЁмҲңнҷ”

6. мқҳлЈҢ мӢңмҠӨн…ң ліҙм•Ҳ мҡ”кө¬мӮ¬н•ӯ
   вҶ’ мһҗлҸҷ к°ңмқём •ліҙ л§ҲмҠӨнӮ№
```

### кІ°лЎ 

**DevOnмқҖ Struts 1.xмқҳ м•„нӮӨн…ҚмІҳлҘј кё°л°ҳмңјлЎң н•ҳлҗҳ,**

1. **MiPlatform м—°лҸҷ**мқ„ мң„н•ң лҚ°мқҙн„° ліҖнҷҳ кі„мёө м¶”к°Җ
2. **EJB м—Ҷмқҙ** м—”н„°н”„лқјмқҙмҰҲ кё°лҠҘ кө¬нҳ„ (PC, TxServiceUtil)
3. **м„ м–ём Ғ м„Өм •**мқ„ нҶөн•ң мғқмӮ°м„ұ н–ҘмғҒ (Requisite, Interceptor)
4. **мқҳлЈҢ мӢңмҠӨн…ң нҠ№нҷ”** кё°лҠҘ м ңкіө (к°ңмқём •ліҙ л§ҲмҠӨнӮ№)

**мқҙлҹ¬н•ң лҸ…м°Ҫм„ұмңјлЎң 2000л…„лҢҖ мӨ‘л°ҳ~2010л…„лҢҖ н•ңкөӯмқҳ м—”н„°н”„лқјмқҙмҰҲ мӢңмҠӨн…ң к°ңл°ңм—җм„ң л„җлҰ¬ мӮ¬мҡ©лҗҳм—ҲмҠөлӢҲлӢӨ.**
