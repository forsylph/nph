# devon-framework.jar

## 1. 목적

이 문서는 `devon-framework.jar`를 reference가 아니라 실제 구조 파악용으로 볼 때 필요한 최소 요약본이다.

## 2. 중요한 이유

- 소스가 없는 클래스가 많다.
- `LCommonDao`, `LQueryMaker`, `LServiceProxy` 같은 핵심 추상화가 jar 쪽에 걸쳐 있다.
- API/Javadoc과 함께 봐야 구조를 닫을 수 있다.

## 3. 무엇을 볼 때 여길 참고하는가

- 클래스는 실존하는데 소스가 없는 경우
- 메서드 표면을 확인해야 하는 경우
- 패키지 분포로 프레임워크 설계를 파악할 때
