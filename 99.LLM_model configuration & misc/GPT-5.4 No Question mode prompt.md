
# chatgpt response

이번 작업 한정으로만 적용되게 하는 문구로 바꿔드리겠습니다. 전역 설정이 아니라 **이번 세션/이번 요청에만 유효한 실행형 프롬프트** 형태가 맞습니다.

맞습니다.  
그럼 **전역 설정 건드리지 말고, 이번 작업 프롬프트 안에만 “묻지 말고 진행” 규칙을 박아 넣는 방식**이 제일 맞습니다.

Codex는 시작 프롬프트를 그대로 작업 기준으로 삼고, 프로젝트의 `AGENTS.md`도 함께 읽지만, **이번 한 번만** 강하게 제어하려면 그냥 **첫 프롬프트를 실행형으로 쓰는 것**이 가장 간단합니다. 공식 문서상 Codex는 초기 user prompt로 세션을 시작하고, 프로젝트 지침 파일은 세션 시작 시 함께 읽습니다. 또 승인 정책과 샌드박스는 설정으로 제어되지만, 작업 방식 자체는 프롬프트로 크게 유도할 수 있습니다. ([OpenAI Developers](https://developers.openai.com/codex/guides/agents-sdk/?utm_source=chatgpt.com))
아래처럼 쓰시면 됩니다.

```text
이번 작업에 한해서만 아래 규칙으로 행동해라.

작업 원칙:
- 중간 질문 없이 진행한다.
- 사소한 애매함은 스스로 가장 보수적이고 안전한 방향으로 판단한다.
- 설명보다 실제 실행과 파일 반영을 우선한다.
- 가이드 문서를 읽기만 하지 말고, 필요한 변경을 직접 적용한다.
- 기존 설정과 파일은 최대한 보존하고, 변경 전 백업이나 diff가 남도록 한다.
- 위험한 작업(대량 삭제, 인증/보안 변경, 외부 전송, 작업 폴더 밖 수정)은 하지 않는다.
- 어떤 단계가 실패하면 멈추지 말고 더 안전한 대안으로 계속 진행한다.
- 완료 가능한 범위까지 끝까지 수행하고, 마지막에만 결과를 보고한다.

최종 보고 형식:
1. 수행한 작업
2. 수정한 파일
3. 실행한 명령
4. 남은 확인 사항
5. 내가 지금 바로 검토할 포인트

이번 작업 목표:
[여기에 실제 목표를 적기]
```

이 프롬프트가 좋은 이유는 단순히 “묻지 마”가 아니라, **질문 대신 무엇을 기준으로 판단할지**를 같이 주기 때문입니다.

사용자님 상황처럼 졸릴 때는 더 짧은 치트 버전도 좋습니다.

```text
이번 작업은 중간 질문 없이 끝까지 진행해라.
사소한 애매함은 스스로 판단하고, 가장 안전한 방법을 선택해라.
설명보다 실행을 우선하고, 기존 파일은 최대한 보존해라.
위험한 작업만 피하고, 실패 시 더 안전한 대안으로 우회해라.
마지막에만 변경 내용, 수정 파일, 실행 명령, 확인 필요 사항을 보고해라.

목표:
[실제 작업]
```

그리고 **가장 잘 먹히는 실전 문장**은 이 한 줄입니다.

```text
문서 요약 모드로 가지 말고 실행 모드로 작업해라. 중간 질문 없이 완료 가능한 범위까지 직접 적용하라.
```

사용자님 용도로 바로 바꿔드리면, 예를 들어 Codex 설정 수정 작업은 이렇게 넣으면 됩니다.

```text
이번 작업에 한해서만 중간 질문 없이 진행해라.

문서 요약이 아니라 실행이 목표다.
가이드 문서를 참고하되, 나에게 다시 묻지 말고 실제 적용까지 수행해라.

해야 할 일:
1. ~/.codex/config.toml 존재 여부 확인
2. 없으면 생성, 있으면 백업 후 최소 수정
3. GPT-5.4 긴 컨텍스트 설정 반영
4. 기존 사용자 설정 최대한 보존
5. 적용 후 검증 명령까지 실행
6. 마지막에만 결과 보고

행동 원칙:
- 사소한 애매함은 스스로 판단
- 위험한 변경 금지
- 작업 폴더 밖 영향 최소화
- 실패 시 멈추지 말고 안전한 대안 선택
- 설명보다 실제 변경 우선

최종 보고:
- 변경 전 상태
- 수정 파일
- 적용한 값
- 실행한 검증 명령
- 내가 나중에 점검할 핵심 3가지
```

추가로, 설정 우선순위는 **CLI `--config` 오버라이드 > 프로필 > 프로젝트 `.codex/config.toml` > 사용자 `~/.codex/config.toml`** 순서라서, 정말 이번 작업 한정으로만 강하게 주고 싶다면 **프롬프트 + 일회성 `--config`** 조합도 가능합니다. ([OpenAI Developers](https://developers.openai.com/codex/config-basic/?utm_source=chatgpt.com))
